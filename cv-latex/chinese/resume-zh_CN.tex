\section{科研经历}
\datedsubsection{\textbf{面向真实数据分析的 Agentic 训练\textsuperscript{‡}} \quad \textit{项目负责人}}{2026.3 - 至今}
现有数据分析 benchmark 多依赖\textbf{固定标准答案}进行评测，难以覆盖开放场景下 data agent 产生的\textbf{非标准但高价值 insight}，从而系统性低估其真实能力。为此，我们构建了\textbf{大规模交互式数据湖环境}，支持 agent 在真实数据分布中自主探索；并基于该环境，在\textbf{24 张 A100}上训练 \textbf{Qwen3.5-9B} 作为 \textbf{verifier agent}，对 data agent 生成的每条 insight 进行\textbf{细粒度的正确性与价值验证}。进一步地，我们将训练后的 verifier 融入\textbf{迭代式 agent harness}，形成``\textbf{生成-验证-反馈}''闭环，在真实业务场景中产出了许多\textbf{高价值 insight}。\\
$\triangleright$~ 入选\textbf{中关村学院深澜计划}，预计投稿 \textit{ICLR 2027}

\datedsubsection{\textbf{DeepRead：文档结构感知的Agentic Search\textsuperscript{†}} \quad \textit{项目负责人}}{2025.10 - 2025.2}
现有 Agentic Search 往往将长文档视为扁平 chunk 集合，忽略文档原生的\textbf{层级结构}与\textbf{顺序逻辑}。我们提出 \textbf{DeepRead}：一种结构感知的文档推理 Agent，利用 OCR 保持版面结构保真，构建\textbf{段落级坐标}，并为 LLM 设计两类协同工具：\textbf{Retrieve}与 \textbf{ReadSection}，使得大模型涌现 ``locate-then-read'' 推理范式，显著缓解传统检索的上下文碎片化问题。在四个覆盖多类型文档的基准上，DeepRead 相比 \textbf{Search-o1} 风格 Agentic Search 基线平均提升 \textbf{10.3\%}。\\
$\triangleright$~ \textbf{Resubmit} \textit{KDD 2027 (CCF-A)}，, 预印本被知名媒体\textbf{新智元}报道

\datedsubsection{\textbf{MuDABench：多文档分析问答基准与多Agent框架\textsuperscript{†}} \quad \textit{项目负责人}}{2025.4 - 2025.12}
\textbf{大规模}文档集合蕴含丰富知识，但在\textbf{上千份}长文档中实现可扩展的\textbf{多文档分析问答}仍具挑战。本项目基于\textbf{远程监督}与专家校验，构建了当前\textbf{规模最大的}多文档分析问答基准之一（\textbf{80k+页}文档、\textbf{332}道分析型问题、\textbf{4,964}结构化中间事实），用于系统评测多文档检索、跨文档聚合与数值推理能力。实验显示，标准检索服务在该设置下表现有限：以 OpenAI \textbf{File Search} 为代表的基线最终准确率最高仅为较低水平；我们提出的\textbf{Multi-Agent}工作流则将其显著提升。进一步误差分析表明，主要瓶颈在于\textbf{单文档信息抽取}的稳定性与高精度定位。\\
$\triangleright$~ \textbf{发表于} \textit{ACL 2026 Findings (CCF-A)}

\datedsubsection{\textbf{基于 SHAP 的公司金融因果推断框架\textsuperscript{*}} \quad \textit{项目负责人}}{2024.9 - 2025.1}
针对传统面板数据模型依赖线性假设、难以刻画复杂非线性动态的问题，我们考虑了编码面板数据来提升模型的拟合能力，同时保持\textbf{统计因果推断}上的解释性。我们构建了可解释建模框架并集成 \textbf{Optuna} 自动超参搜索，在TFP预测任务上取得 $R^2=0.76$，有效超越传统线性基线；进一步使用 \textbf{SHapley Additive exPlanations (SHAP)} 对面板数据中的机制进行可追溯分析，量化并分解 ESG 评级分歧对结果变量的\textbf{非线性边际贡献}，从而揭示潜在因果路径。\\
$\triangleright$~ \textbf{发表于} \textit{Finance Research Letters}（中科院 Q1 Top，JCR Q1，IF: 6.9），\href{https://mp.weixin.qq.com/s/5oEEnyVM_0PdWTg-7X8RJA}{\textbf{清华大学因果推断研讨会优秀论文（Top 3\%）}}
