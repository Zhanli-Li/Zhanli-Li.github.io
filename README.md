# Zhanli Li Personal Website

This repository contains the source code for [zhanli-li.github.io](https://zhanli-li.github.io).

## Structure

- `_pages/`: main pages such as home, CV, publications, and blog archive
- `_posts/`: blog posts
- `_publications/`: publication entries
- `_talks/`: talks and presentations
- `files/`: static files such as PDFs and figures

## Local Preview

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open `http://localhost:4000`.
